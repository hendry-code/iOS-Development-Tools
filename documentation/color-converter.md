# Color Converter

## Overview
The **Color Converter** is a universal color conversion utility that instantly translates colors between multiple formats used across iOS, Android, and Web development.

## Features
- **Universal Input**: Accepts HEX, RGB, RGBA, HSL, or HSLA with auto-detection
- **Visual Color Picker**: Native HTML5 color picker for intuitive selection
- **RGB Sliders**: Fine-tune Red, Green, and Blue channels individually
- **Alpha Control**: Adjust transparency with a dedicated opacity slider
- **13 Output Formats**: Instantly view the color in all formats with one-click copy
- **Contrast Detection**: Automatically shows whether text should be white or black

## Supported Formats

| Platform | Formats |
|----------|---------|
| **Universal** | HEX, HEX with Alpha |
| **Web** | RGB, RGBA, HSL, HSLA, CSS Variable, Tailwind (closest match) |
| **iOS** | UIColor (UIKit), Color (SwiftUI) |
| **Android** | Color Int (0xAARRGGBB), XML Resource, Jetpack Compose Color |

## Usage
1. **Enter a color** in any format (e.g., `#3B82F6`, `rgb(59, 130, 246)`, `hsl(217, 91%, 60%)`)
2. **Or use the picker** to visually select a color
3. **Adjust alpha** if needed using the opacity slider
4. **Copy any format** by clicking the copy button next to each output
