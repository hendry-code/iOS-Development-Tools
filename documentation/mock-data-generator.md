# Mock Data Generator

## Overview
The **Mock Data Generator** allows you to create large datasets of realistic test data for development and testing. You can define a schema with specific data types and nest objects to match your application's data model.

## Features
- **Schema Builder**: Visually define your data structure.
- **Rich Data Types**: Supports names, emails, UUIDs, booleans, dates, and more.
- **Nested Objects**: Create complex, deep JSON structures.
- **Flexible Output**: Generate data in **JSON** or **CSV** format.
- **Real-Time Preview**: see sample data update instantly as you modify the schema.

## Usage
1. **Define Schema**:
   - Add fields using the schema builder.
   - Select types (e.g., `uuid`, `name`, `full_name`).
   - Create nested objects by adding children to 'object' type fields.
2. **Configure**: Set the number of rows (e.g., 100) and the output format (JSON/CSV).
3. **Generate**: The tool automatically generates the data.
4. **Copy/Download**: Copy the result from the preview block.
