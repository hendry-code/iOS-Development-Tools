# File Editor

## Overview
The **File Editor** is a specialized text editor for quick modifications to your localization files without leaving the dashboard. It features a "Find & Replace" tool tailored for editing large translation files.

## Features
- **Direct Editing**: Open and edit `.strings`, `.xml`, `.json`, `.properties`, and `.xcstrings` files.
- **Find & Replace**:
    - Search for text within the file.
    - Replace individual occurrences or all matches at once.
    - Visual highlighting of search terms.
- **Download**: Save your changes by downloading the modified file.

## Usage
1. **Open File**:
   - Upload a file to start editing.
2. **Edit**:
   - Type directly in the editor pane.
   - Use `Find & Replace` (magnifying glass icon) to make bulk changes.
3. **Save**:
   - Click the **Download** icon to save your changes to a new file.
