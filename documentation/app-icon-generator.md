# App Icon Generator

## Overview
The **App Icon Generator** simplifies the creation of iOS app icon assets. It takes a single high-resolution image and automatically generates the entire set of required icon sizes for iPhone, iPad, watchOS, and macOS.

## Features
- **One-Click Generation**: Upload one master image (1024x1024 recommended) and get everything you need.
- **Platform Selection**: Toggle specific platforms (iPhone, iPad, Watch, Mac) to generate only what you need.
- **Asset Catalog Ready**: Generates a standard `AppIcon.appiconset` folder structure with a valid `Contents.json` file.
- **ZIP Download**: Delivers the result as a ready-to-import ZIP archive.

## Usage
1. **Upload Master**: Drop a 1024x1024 PNG or JPG image.
2. **Select Platforms**: Check the boxes for the devices you support.
3. **Generate**: Click **Generate & Download ZIP**.
4. **Import**: Unzip the file and drag the `AppIcon.appiconset` folder into your Xcode `Assets.xcassets`.
